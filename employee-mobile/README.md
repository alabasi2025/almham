# تطبيق أنظمة العباسي المتخصصة للموظفين

تطبيق Android للموظفين يعمل مع نظام العباسي الرئيسي.

## ما يفعله حالياً

- تسجيل دخول عبر API النظام الرئيسي.
- تسجيل حضور وانصراف مع GPS.
- تشغيل تتبع الخلفية أثناء الوردية فقط.
- حفظ نقاط الموقع والطلبات محلياً عند انقطاع الإنترنت.
- مزامنة الطلبات المحفوظة عند عودة الاتصال.

## التشغيل المحلي

```bash
cd employee-mobile
npm install
cp .env.example .env
npm run start
```

للأندرويد Emulator استخدم:

```env
EXPO_PUBLIC_API_URL=http://10.0.2.2:3000
```

للجوال الحقيقي على نفس الشبكة، استخدم IP جهاز السيرفر:

```env
EXPO_PUBLIC_API_URL=http://192.168.1.50:3000
```

## ملاحظات مهمة

- التتبع الخلفي الحقيقي يحتاج Android build، وليس مجرد فتح المتصفح.
- التطبيق لا يسجل أي موقع قبل الحضور أو بعد الانصراف.
- في الإنتاج يجب تشغيل السيرفر عبر HTTPS.
- لا تضع أسراراً في `EXPO_PUBLIC_*` لأنها تظهر داخل تطبيق الجوال.
