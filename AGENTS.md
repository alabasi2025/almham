## 2. 🎯 ما هذا المشروع

**الاسم:** نظام الهام الموحّد  
**العميل:** شركة الهام للكهرباء — 4 محطات في اليمن  
**الهدف:** ERP موحّد يحل 10 نقاط ألم (راجع `business-context.md §13`)

**ليس هذا:**
- ❌ نظام ECAS/YemenSoft (قديم، يُستخدم كمرجع بيانات فقط)
- ❌ إصلاح لمشكلة "spy detection" (انحراف سابق، انتهى)
- ❌ تكامل إلزامي مع Hexcell (يأتي في موجة 2-3)

## 3. 🏗️ التقنيات (ثابتة — لا تغيّرها بدون طلب صريح)

| الطبقة | التقنية | الإصدار |
|--------|---------|---------|
| Frontend | Angular | 21 (standalone + signals) |
| UI | Angular Material + Chart.js + Leaflet | أحدث متوافق |
| Backend | Hono | 4.x |
| ORM | Drizzle | 0.38 |
| DB | PostgreSQL | 15+ |
| Node | Node.js | 22 LTS |

**قواعد الكود:**
- ✅ Standalone components فقط (لا NgModules)
- ✅ Signals للحالة (لا BehaviorSubject جديد)
- ✅ `inject()` للخدمات (لا constructor injection قديم)
- ✅ Lazy loading لكل مسار
- ✅ Tailwind-style utility مسموح، لكن SCSS هو الأساس
- ✅ UUID للـ IDs (`defaultRandom`)
- ✅ enums في DB عبر `pgEnum`
- ✅ كل `timestamps` بـ `timestamp('created_at').defaultNow().notNull()`

## 5. 🌍 اللغة والتوطين

- **كل المحادثة مع المستخدم بالعربية**
- **رسائل API بالعربية** (مثلاً: `{ error: 'المحطة غير موجودة' }`)
- **أسماء الجداول/الأعمدة بالإنجليزية** (`stations`, `meter_readings`)
- **UI labels بالعربية** (`العدّادات`, `التحصيل اليومي`)
- **التاريخ:** ميلادي + `ar-YE` locale للعرض
- **العملة الأساسية:** ريال يمني (YER)

## 6. 🛡️ القواعد الذهبية

### ✅ افعل
- اقرأ `business-context.md` أولاً
- اقترح قبل التنفيذ للتغييرات الكبيرة (schema جديد، حذف كود)
- أضف migrations واضحة لكل تعديل DB
- اكتب seeds للبيانات الحقيقية (المحطات الأربع، 16 موظف بالاسم)
- اختبر بـ `npm run db:push` ثم تشغيل الـ backend ثم frontend
- حدّث `business-context.md §سجلّ التعديلات` بعد كل جلسة

### ❌ لا تفعل
- لا تبنِ مميزات لم تُطلب
- لا تكتب تعليقات كثيرة في الكود (الاسم الوصفي كافٍ)
- **لا تكتب كلمات سر أو API keys في الكود** (استخدم `.env`)
- لا تُضف dependencies ثقيلة بدون إذن
- لا تحذف ملفات قديمة بدون تأكيد
- لا تذهب في رحلات استكشافية طويلة (مثل ECAS spy-detection)
- لا تكتب باللغة الإنجليزية مع المستخدم
